SBX 5K

Shadowmatic Labs · 2020 · PC · FPS

Project Overview

SBX 5K is a high-speed FPS game developed during the Covid pandemic. The game features a fast paced 1v1 gameplay, where players build the map throughout the match to provide more fun.

My Role

Unity game developer, responsible for implementing the matchmaking system, lobbies, chatting system and authoritative real-time FPS logic

Game Features

  • Quake like gameplay, for 1v1 matches
  • In-match map building
  • Inventory with multiple weapons
  • Overall leaderboard for increased competition

Technical Implementation

  • Matchmaking system using Photon & Playfab
  • Lobbies & Friends list using Photon Chat & Playfab
  • Authoritative server logic to prevent cheating

Development challenges

SBX 5K gathered a few thousand players, having hit around 100 consecutive users, playing at the same time during events. The main challenge was balancing the fast-paced gameplay with the map-building mechanics to create a unique and engaging experience.
