SBX 5K

Game Jam Entry · 2020 · PC/Mobile · Racing

Project Overview

SBX 5K is a high-speed racing game developed during a 48-hour game jam. The game features intense track-based racing with power-ups, dynamic obstacles, and a focus on achieving the perfect lap time.

My Role

Lead programmer and game designer responsible for the core racing mechanics, physics system, track design tools, and mobile optimization within the tight jam timeframe.

Game Features

  • Arcade-style racing physics with drift mechanics
  • Multiple track layouts with dynamic obstacles
  • Power-up system for competitive gameplay
  • Time trial and multiplayer racing modes
  • Cross-platform deployment (PC and mobile)

Technical Implementation

  • Custom vehicle physics controller optimized for mobile
  • Spline-based track creation system
  • Efficient rendering pipeline for 60fps on mobile devices
  • Networked multiplayer using Unity's networking
  • Adaptive quality settings for different hardware

Game Jam Results

SBX 5K placed in the top 10% of submissions and received praise for its smooth controls and polished feel despite the short development time. The project demonstrated my ability to deliver a complete, playable experience under pressure.
