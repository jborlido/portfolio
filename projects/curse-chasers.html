Curse Chasers

Independent Project · 2019 · PC · Action-Adventure

Project Overview

Curse Chasers is an action-adventure game that combines fast-paced combat with exploration mechanics. Players navigate through mysterious environments while battling supernatural enemies and uncovering ancient secrets.

My Role

Solo developer responsible for all aspects of game development including gameplay programming, level design, character controller implementation, and visual effects systems in Unity.

Key Features

  • Fluid character movement and combat system
  • Dynamic lighting and atmospheric effects
  • Procedural enemy AI behavior
  • Custom shader work for mystical visual effects
  • Modular level design system

Technical Highlights

  • Custom character controller with wall-running mechanics
  • State machine-based AI for multiple enemy types
  • Shader programming for ethereal visual effects
  • Performance optimization for complex particle systems

Development Insights

This project taught me valuable lessons about scope management and the importance of player feedback. The combat system went through several iterations based on playtesting, resulting in a more responsive and satisfying player experience.
