Curse Chasers

Independent Project · 2019 · Google Play · App Store · Turn-Based Strategy

Project Overview

Curse Chasers is an action-adventure game that combines fast-paced combat with exploration mechanics. Players navigate through mysterious environments while battling supernatural enemies and uncovering ancient secrets.

My Role

Solo developer responsible for all aspects of game development including gameplay programming, backend (matchmaking and battle communication) and performance optimizations.

Key Features

  • Turn-based multiplayer/PVE combat
  • Tile board character movement
  • Procedural enemy AI behavior
  • Contextual UI for in-game instructions

Technical Highlights

  • Custom server for processing matchmaking and battles
  • State machine-based AI for PVE enemies
  • Performance optimization for wide range of devices

Development Insights

This project taught me valuable lessons about scope management and the importance of player feedback. The combat system went through several iterations based on playtesting, resulting in a more responsive and satisfying player experience.
